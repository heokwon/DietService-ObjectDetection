### Introduction
* 현재 자기관리가 하나의 트랜드처럼 자리잡고있음
* 운동만큼 식단의 중요성이 부각됨
* 핸드폰 카메라 하나만으로 식단관리가 가능한 서비스가 있으면 어떨까 하는 생각에 시작한 프로젝트
* Object Detection을 통해 음식을 인식하고 칼로리와 영양정보를 제공
### Models and Data

* 총 100개의 class
* multiple food-item, single food-item으로 구성
